Features Included in Project

  1. Output Generation (For both AVs and HVs)
    1. CSVs (Comma Separated Values)
      1. Values: Speed, Safe Distance, Reaction Time, Stopping Time
    2. Histograms Plots
      1. Safe Distance
      2. Speed
      3. Deceleration Force
      4. Reaction Time
      5. Stopping Time
    3. Normal Distribution Plot
      1. Safe Distance
    4. Scatter Plot
      1. Stopping Time
    5. Experiment Summary (PDF and PNG with plots)
      1. Congestion
      2. No. of Collisions
      3. Avg. Braked Cars per min
      4. Avg. Safe Distance
      5. Capacity (Cars/min)
      6. Total Simulation Time
  2. Documentation (How the System Works)

RN - Reservation Nodes

TL - Traffic Lights

CAwSD4WI - Collision Avoidance with Safe Distance and 4 Way Intersection


Congestion: Amount of cars per minute around an intersection or the number of collisions prevented.

Collision: The number of collisions that occurred.

Braked Cars per Min: The average number of cars braked every minute.

Average Safe Distance (AV): The average safe distance every second for AVs .

Average Safe Distance (HV): The average safe distance every second for HVs .

Capacity AV - car(s) / min: The average number of AV cars that reach the destination every minute.

Capacity HV - car(s) / min: The average number of HV cars that reach the destination every minute.

Total Simulation Time (s): Total time of simulation for each experiment method.
