How to Use

  1. To Compile Simulation
    1. Go to the top level of the project folder of the project in PyCharm (AVHV_Main) and run tests/test.py. You can change simulation settings like time length in AVHV_Main/tests/test.py and route combination and number of cars in AVHV_Main/tests/route_catalogue.py although these (route combination and number of cars) have been set to be produced randomly at compile time. Depending on the length of time the simulation is set for and the number of cars involved, this compilation may take any where from a few seconds to a few hours - this is not a bug but a feature.
  2. To Run Simulation
    1. Go to the top level of the project folder AVHV_Main and open viewer.html in your browser. Alternatively, go directly here. Press Spacebar to start the simulation. You can also use the left and right arrow keys to go to previous and next frame respectively.
  3. To View Experiment Summary
    1. On the viewer, click the "Experiment Summary (PDF)" link at the bottom right corner. From the project folder, you can also find the experiment summary file at AVHV_Main/output/Experiment_Summary.pdf. The plots representing the recorded values in the summary file can be found in the AVHV_Main/output/Experiment_Summary.png
  4. To View Simulation Outputs
    1. Go to AVHV_Main/output/AVHV_Traffic_Lights for Experiment run with Traffic Lights.
    2. Go to AVHV_Main/output/AVHV_CAwSD4WI for Experiment run with Car Avoidance with Safe Distance with 4 Way Intersection.
    3. Go to AVHV_Main/output/AVHV_Reservation_Nodes for Experiment run with Reservation Nodes.
    See the list of outputs included in Features/Inclusions
  5. To View Documentation
    1. Go to How The System Works (Online Documentation)
